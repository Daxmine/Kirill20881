GIT Flow

Git clone - для того , щоб "склонить " репозиторий

Git pull - для того , щоб отримати из репозетории

Git checkout - b branchname - создать нову ветку и переключитись на неї

Git add - добавить изметнения,которые сделали локально

Git commit -m "commit text" - подписать их

Git push origin branchname - запушить на git сервере
